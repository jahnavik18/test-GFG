<h2><a href="https://www.geeksforgeeks.org/problems/magic-square/1">Magic Square</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given a&nbsp;<strong>n*n&nbsp;</strong>matrix&nbsp;<strong>mat[][]&nbsp;</strong>, check whether the matrix is a <strong>magic square&nbsp;</strong>or not.<br></span><span style="font-size: 14pt;">A Magic Square is a n x n matrix of distinct elements from 1 to n * n</span><span style="font-size: 14pt;"> where the sum of any row, column, or diagonal is always equal.&nbsp;</span></p>
<p><strong><span style="font-size: 14pt;">Examples:</span></strong></p>
<pre><strong><span style="font-size: 14pt;">Input: </span></strong><span style="font-size: 14pt;">mat[][] = [[2, 7, 6], <br>                 [9, 5, 1], <br>                 [4, 3, 8]]<br><strong>Output: </strong>true<br><strong>Explanation: </strong>Sum of rows, columns, and diagonals are all equal to 15. </span></pre>
<pre><strong><span style="font-size: 14pt;">Input:</span></strong><span style="font-size: 14pt;"> mat[][] = [[1, 2], <br>                 [3, 4]]<br><strong>Output: </strong>false<br><strong>Explanation: </strong>Sum of all rows and columns are not same.</span></pre>
<pre><strong><span style="font-size: 14pt;">Input:</span></strong><span style="font-size: 14pt;"> mat[][] = [[1, 1, 1], <br>                 [1, 1, 1],<br>                 </span><span style="font-size: 14pt;">[1, 1, 1]]<br><strong>Output: </strong>false<br><strong>Explanation: </strong>All sums are same but all elements from 1 to n<sup>2</sup> are not present.<br></span></pre>
<p><strong><span style="font-size: 14pt;">Constraints:&nbsp;<br></span></strong><span style="font-size: 14pt;">1&nbsp;</span><span style="font-size: 18.6667px;">≤</span><span style="font-size: 14pt;"> n </span><span style="font-size: 18.6667px;">≤</span><span style="font-size: 14pt;"> 10<sup>3</sup></span></p>
